# Contributing

Thank you for helping improve RaeburnAI Business Twin.

## Local development

```bash
pnpm install
pnpm dev
```

## Quality checks

```bash
pnpm typecheck
pnpm test
pnpm build
```

## Pull requests

Please include:

- Clear description of the change.
- Tests for domain or API changes.
- Documentation updates when behaviour changes.

## Good first contribution areas

- Document parsers.
- Supabase storage adapter.
- Scenario templates.
- Workflow visualisation.
- KPI importers.
